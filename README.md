# Python e Flask

O projeto consiste em um sistema de controle de estoque, onde poderemos criar, editar e excluir produtos. Além disso, tem uma página de login e comandos flask.

## Ferramentas utilizadas

* Python
* Flask
* MongoDB Atlas
